# ✨ About

Fresh N Delight is a premium Shopify eCommerce website designed for a modern freeze-dried fruit brand. The project focuses on creating a clean shopping experience with responsive layouts, attractive product displays, fast performance, and an intuitive user interface.

Built using Shopify's modern theme architecture, the store provides customers with a smooth browsing experience while showcasing premium-quality products through elegant design and optimized functionality.

# ⚡ Installation

Clone the repository

```bash
git clone https://github.com/pujanshekhda/fresh-n-delight-shopify.git
```

Go to the project

```bash
cd fresh-n-delight-shopify
```

Run Shopify Theme

```bash
shopify theme dev
```

Deploy Theme

```bash
shopify theme push
```
